# E-Commerce Sales Performance Dashboard

## Project Overview
This project analyzes e-commerce sales data to uncover trends, profitability, and regional performance using Excel and Power BI.

## Tools Used
- Excel (Power Query, formulas)
- Power BI (DAX, interactive visuals)

## Key KPIs
- Total Sales
- Total Profit
- Average Profit Margin
- Quantity Sold

## Dashboard Features
- Region, Category, and Date slicers
- Profitability by product & category
- Sales and profit trend analysis
- Top products by revenue

## Files
- `e_commerce.xlsx` → cleaned dataset & calculations
- `e_commerce_project.pbix` → interactive dashboard

## Insights
- Identified high-margin categories and low-performing regions
- Detected seasonal sales trends
- Highlighted top revenue-driving products
